# Запуск проекта

## 1. Проверка Java

```bash
java -version
```

Требуется Java 21.

## 2. Сборка и тесты

Windows PowerShell:

```powershell
.\mvnw.cmd clean test
```

Linux / macOS:

```bash
./mvnw clean test
```

## 3. Запуск приложения

Windows PowerShell:

```powershell
.\mvnw.cmd spring-boot:run
```

Linux / macOS:

```bash
./mvnw spring-boot:run
```

## 4. Проверка адресов

- `GET /api/greet/World`
- `GET /api/students`
- `POST /api/students`
- `GET /students`
- `/h2-console`

## 5. Примеры curl

```bash
curl http://localhost:8080/api/greet/World

curl -u user:password http://localhost:8080/api/students

curl -u admin:password -X POST http://localhost:8080/api/students   -H "Content-Type: application/json"   -d '{"name":"Sash","surname":"Brown"}'

curl -u admin:password -X PUT http://localhost:8080/api/students/1   -H "Content-Type: application/json"   -d '{"name":"Deema","surname":"Johnson"}'

curl -i -u admin:password -X DELETE http://localhost:8080/api/students/1
```
